# 美元流动性框架介绍

这份文档供模型在分析每日数据前阅读，用于建立统一的美元流动性理解框架。

## 1. 分层框架

美元流动性不是单一利率，而是一组从央行资产负债表传导到市场融资价格的链条：

1. Fed负债端：TGA、RRP、流通现金和其他负债会影响银行准备金水位。
2. 银行准备金：准备金越充裕，银行间资金价格越不容易上行；准备金越稀缺，EFFR更容易靠近或突破政策锚。
3. 银行间融资：EFFR和OBFR观察无抵押隔夜美元资金成本。
4. 回购和抵押品融资：SOFR、TGCR、BGCR观察以国债为抵押的融资压力。
5. 国债吸收能力：拍卖需求、repo fails和交易商资产负债表压力会影响国债融资链条。
6. 离岸美元：当离岸美元融资紧张时，美元指数、跨币种融资成本和新兴市场压力可能同步上升。
7. 债券定价锚：1Y/3Y/10Y美债收益率、10Y实际收益率和10Y-2Y、10Y-3M利差观察近端政策路径、中段再定价、真实贴现率、增长预期、期限溢价和资产定价锚。
8. 信用与证券市场：IG/HY OAS、VIX和NFCI观察利率压力是否进入企业融资成本、股票波动率、风险偏好和广义金融条件；高盛FCI本身是专有指标，无免费官方API时只能用公开代理。

## 2. 核心变量

- TGA（财政部一般账户）：上升通常抽走银行准备金，下降通常释放准备金。
- RRP（隔夜逆回购）：下降可能代表货币基金资金回到市场，是缓冲释放；但余额过低后，继续缓冲QT和TGA补库的能力下降。
- SOMA（美联储持仓）：反映QT背景，通常是周频结构性变量，不应当当作昨日边际信号。
- EFFR（有效联邦基金利率）：银行间无抵押隔夜资金价格，是观察准备金边际稀缺的重要指标。
- OBFR（隔夜银行融资利率）：更广义的银行隔夜融资价格，观察压力是否扩散。
- SOFR（担保隔夜融资利率）：以美国国债为抵押的回购融资价格，观察抵押融资压力。
- TGCR（三方一般抵押品利率）：三方回购市场一般抵押品融资成本。
- BGCR（广义一般抵押品利率）：覆盖更广的一般抵押品回购融资成本。
- IORB（准备金余额利率）：美联储支付给银行准备金的利率，是银行持有准备金的政策锚。
- 1Y Treasury Yield（1年期美国国债收益率）：观察未来一年近端政策路径和短端再定价。
- 3Y Treasury Yield（3年期美国国债收益率）：观察政策路径从短端向中段扩散的再定价。
- 10Y Treasury Yield（10年期美国国债收益率）：全球资产折现率和长期美元融资价格的重要锚。
- 10Y Real Yield（10年期TIPS实际收益率）：真实贴现率，对成长股、黄金和长期资产估值更敏感。
- 10Y-2Y / 10Y-3M Treasury Spread（长短端美债利差）：观察收益率曲线倒挂、增长预期和降息预期。
- IG OAS / HY OAS（投资级/高收益债期权调整利差）：观察企业融资风险溢价是否扩大。
- VIX（标普500隐含波动率指数）：观察证券市场风险偏好和避险需求。
- NFCI（芝加哥联储全国金融条件指数）：公开金融条件代理，正值偏紧，负值偏松。

## 3. 日元 Carry Trade 叠加框架

日元 carry trade 的核心不是单纯看 USD/JPY，而是看“融资成本低、收益端高、汇率波动低、风险偏好强、仓位不拥挤”这几个条件是否同时成立；反过来，任何一环恶化都可能触发去杠杆和日元空头回补。

### 3.1 核心监控变量

- 美日短端利差：JPY 隔夜融资成本（BOJ 无担保隔夜拆借利率、TONA/政策锚）相对 USD 短端利率（SOFR、EFFR、IORB、2Y UST）。利差越宽，carry 激励越强；利差快速收窄，平仓压力上升。
- 美日长端利差：10Y UST - 10Y JGB、2Y UST - 2Y JGB。长端利差影响日债资金回流和海外债券持仓吸引力。
- 汇率趋势与波动：USD/JPY 点位、1日/5日/20日变动、实现波动率。日元快速升值比缓慢贬值更容易触发止损和平仓。
- 日本政策预期：BOJ 加息、缩表、购债减少、收益率曲线正常化预期；JGB 10Y/30Y 上行会提高日本本土资产吸引力。
- 全球风险偏好：VIX、信用利差、NFCI、美元指数、股指回撤。风险偏好恶化时，carry trade 更容易从“收益策略”变成“融资回补风险”。
- 仓位拥挤度：CFTC 日元期货非商业净头寸/杠杆基金净头寸、净空头占未平仓合约比例、52周/3年分位。拥挤空头越高，反转越非线性。
- 日本对外投资与资金回流：日本投资者海外债券/股票买卖、经常账户、寿险/银行海外债券配置。数据频率较低，只作结构背景。
- 实际有效汇率：JPY REER/NEER。极端低估本身不触发平仓，但会提高政策干预、资金回流和均值回归敏感度。

### 3.2 优先 API 来源

- BOJ Time-Series Data Search API：日本无担保隔夜拆借利率 `FM01/STRDCLUCON`，USD/JPY 东京市场汇率 `FM08/FXERD04`，JPY NEER/REER `FM09/FX180110001,FX180110002`。端点形如 `https://www.stat-search.boj.or.jp/api/v1/getDataCode?format=json&lang=en&db=FM08&code=FXERD04`，无需鉴权。
- 日本财务省 MOF CSV：JGB 1Y/2Y/5Y/10Y/20Y/30Y/40Y 日度收益率，`https://www.mof.go.jp/english/policy/jgbs/reference/interest_rate/historical/jgbcme_all.csv`，无需鉴权。
- CFTC Public Reporting API：JPY 期货 COT 仓位，Legacy Futures Only 端点 `https://publicreporting.cftc.gov/resource/6dca-aqww.json`，日元合约代码 `097741`，周频、周二数据、周五发布。
- FRED/NY Fed：美元端 SOFR、EFFR、IORB、2Y/10Y UST、美元指数、VIX、信用利差；本框架已有部分接入。
- BIS SDMX API：JPY NEER/REER、全球流动性和跨境银行统计；频率低于日度市场数据，适合作为估值和结构背景。
- 商业备选：Alpha Vantage/Polygon/Twelve Data 可补 USD/JPY 实时或日内；Bloomberg/Refinitiv/CME/ICE 更适合 FX options implied vol、cross-currency basis 和期权偏度，但通常无免费官方 API。

## 4. 判断原则

1. 先检查数据新鲜度：日频数据优先，周频数据只做背景，事件型数据要标注日期。
2. 再判断边际方向：看最新值相对上一期是上升、下降还是持平。
3. 最后判断传导范围：压力是否只在Fed负债端，是否进入银行间融资、回购融资、国债吸收、离岸美元、信用市场或日元carry去杠杆链条。
4. 不要用单个指标下结论：例如TGA下降偏松，但若RRP已接近低位，后续缓冲能力仍可能下降；USD/JPY上涨也不等于carry健康，必须同时看波动、利差和仓位。
5. 对缺失数据要明确说明，不要把缺失当作中性。

## 5. 常见解释

- EFFR低于政策锚：银行间资金不紧，准备金仍有余裕或分布压力不明显。
- EFFR接近或高于政策锚：准备金边际稀缺或分布不均，需要警惕。
- SOFR低于政策锚：回购融资成本不高，抵押融资压力不明显。
- SOFR上升并高于政策锚：回购市场融资压力增强。
- SOFR-EFFR扩大：抵押融资链条相对银行间无抵押融资更紧。
- TGA大幅上升：财政现金余额增加，短期抽走银行准备金。
- RRP接近低位：过去可释放的缓冲垫减少，后续冲击更容易影响准备金。
- 1Y收益率高位：近端政策路径维持高位，现金和短债收益对风险资产形成分流。
- 10Y上行：长期名义折现率和融资成本上升，可能压制风险资产估值。
- 10Y实际收益率上行：真实无风险收益抬升，对成长股、黄金和长久期资产压力更直接。
- 10Y-2Y或10Y-3M倒挂加深：市场更强地定价未来增长放缓或政策降息。
- IG/HY OAS扩大：企业信用风险溢价上升，说明压力传导到债券信用市场。
- VIX上行：证券市场避险需求上升，风险偏好下降。
- NFCI转正或上行：公开金融条件代理显示整体金融条件收紧；但它不是高盛FCI。
